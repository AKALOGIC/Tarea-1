Lenguajes basicos de Programación Web

HTML (Hypertext Markup Language) es un lenguaje de marcado utilizado para crear la estructura y contenido de páginas web, utilizando etiquetas para definir elementos y su presentación.

JavaScript es un lenguaje de programación de alto nivel, interpretado, orientado a objetos y multiparadigma, utilizado principalmente para crear interactividad en páginas web.

CSS es un lenguaje de hojas de estilo utilizado para definir la presentación y el diseño de documentos , permitiendo controlar la apariencia visual de los elementos en una página web.
